HTML

HTML, singkatan dari HyperText Markup Language, adalah bahasa markup dasar yang digunakan untuk membangun struktur dan konten sebuah halaman web. HTML menentukan makna dan struktur konten, sedangkan teknologi lain seperti CSS dan JavaScript digunakan untuk presentasi dan fungsionalitas

PHP

PHP adalah singkatan dari "Hypertext Preprocessor," sebuah bahasa pemrograman open-source yang banyak digunakan untuk membangun aplikasi web dinamis dan interaktif. PHP dijalankan di sisi server dan menghasilkan output yang bisa ditampilkan oleh browser. PHP sering digunakan bersama HTML, CSS, dan JavaScript untuk membuat halaman web yang dinamis dan interaktif

CSS

CSS, singkatan dari Cascading Style Sheets, adalah bahasa stylesheet yang digunakan untuk menjelaskan tampilan atau desain dari halaman web yang dibuat dengan HTML. CSS mengatur berbagai aspek visual seperti warna, font, tata letak, dan lain-lain, membuat halaman web lebih menarik dan mudah dibaca
